# Kanban Task Manager

## Descripció
Aplicació web de gestió de tasques tipus "Kanban" desenvolupada amb HTML, CSS i JavaScript.

## Funcionalitats principals
- Gestió de tasques amb títol, descripció, prioritat i data límit
- Organització de tasques en columnes: Per fer, En curs, Fet
- Filtres i cerca de tasques
- Estadístiques de progrés
- Persistència local (localStorage)
- Disseny responsiu per a diferents dispositius

## Estructura del projecte

projecte-kanban/
├── index.html # Pàgina principal
├── css/
│ └── estils.css # Estils CSS de l'aplicació
├── js/
│ └── script.js # Lògica JavaScript
├── img/ # Imatges i icones
└── README.md # Documentació

## Instal·lació i ús
1. Clona o descarrega el repositori
2. Obre el fitxer `index.html` al teu navegador
3. Comença a afegir tasques!
